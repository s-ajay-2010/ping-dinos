# PING-DINOS
This is a slack-bot of mine:)

## Features
- Add people to your specified channel, when you send /ping-dinos in the channel.
- Send cat facts when you send /ping-dinos-cat-facts in the channel.
- Send dino jokes when you send /ping-dinos-joke in the channel.
- For help message send /ping-dinos-help.

P.S: As I could not test the adding people to my channel feature, so it may not work, but please don't reject hrs due to it:pray:

## Setup
``` bash
git clone https://github.com/s-ajay-2010/ping-dinos.git
cd ping-dinos
cp .env.example .env #add your own tokens from slack-dev & your channel-details
npm install
node index.js
``` 
add the app to your channel and the channel from where you want to message to.
