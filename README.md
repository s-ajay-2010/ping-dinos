# PING-DINOS
This is a slack-bot or mine which adds the people who react with :ajay-pat:(copy and paste that in emoji search in slack) to my private hackclub channel.

## Setup
``` bash
git clone https://github.com/s-ajay-2010/ping-dinos.git
cd ping-dinos
cp .env.example .env #add your own tokens from slack-dev
node index.js
``` 
add the app to your channel and run "/ping-dinos" to make it send msg to which people can now react to.
